# 中文 X Post Skill v1.3 验证说明

- Skill：`zh-x-post@1.3.0`
- 产品流程：单稿生成 → 独立质检 → 不合格时修复一次
- Studio 对用户展示：一篇正文
- 产品内 benchmark：已移除
- Auto rollout：保持关闭

## 迭代依据

- v1.1 第 1 轮：30 分，愿意发布 0/15；主要问题为说教、陈旧和微型作文。
- v1.2 第 2 轮：加入 40 条近期正反语料后，内容从说教变成二手资讯编辑，仍未达到发布预期；还出现把原作者经历转成用户第一人称的问题。
- v1.3：使用用户亲自修改并确认的 AI/Web3 定稿作为黄金样本，学习本人信号、口头思路轨迹和具体记忆，不学习字面模板。

## v1.3 自动回归

仓库检查：

- 素材归属、外部来源和允许叙述人称；
- 重复开头、英文切换、反问、自我修正、口语情绪和记忆锚点；
- 保留 `or`、“一样的”和“歪脖山”等原始表达；
- 拦截人味丢失、过度抛光、凭空情绪和口语模板；
- 拦截新增数字、实体、经历、确定性、伪步骤和内容农场腔；
- Studio 单稿流程、Auto 隔离和 1.3.0 版本归因；
- 40 条研究语料的数量、标签、作者覆盖和短摘要边界。

自动测试不能证明传播效果，也不能替代真实发布判断。
