# 中文 X Post Skill v1.1 验证说明

- Skill：`zh-x-post@1.1.0`
- 产品流程：单稿生成 → 独立质检 → 不合格时修复一次
- Studio 对用户展示：一篇正文
- 产品内 benchmark：已移除
- Auto rollout：保持关闭

## 自动回归

仓库保留 60 份素材 fixtures，用于检查：

- 六类内容路由是否正确；
- 主张和确定性是否保留；
- 是否新增数字、实体或第一人称经历；
- 是否命中模板腔、说教腔和扩写上限；
- 单稿流程是否只生成一篇正文。

这些测试是开发回归，不是用户需要操作的产品功能，也不直接证明内容有传播效果。

## 人工审核

真实内容质量通过每轮 12—15 篇成稿审核：

1. 审核者给出整体分数；
2. 标记最好和最差的编号；
3. 归纳共同问题；
4. 记录真实发布意愿；
5. 只把稳定出现的问题写回 Skill。

第 1 轮内容见 `docs/reviews/中文-X-Skill-v1.1-人工审核第1轮.md`。

## 当前判断

v1.1 已完成结构重构，但传播效果尚未经过真实发布数据验证。下一步依据人工审核修改 Skill，再用实际发布后的浏览、回复、收藏、转发和关注数据判断是否有效。
